## 开始

```
npm run bootstrap, Don't use cnpm install.
```

### 启动开发服务器 并自定义端口

```
npm run dev --port = 4396
```

### 构建预发布环境

```
npm run build:stage
```

### 构建生产环境

```
npm run build
```

### 预览发布环境效果

```
npm run preview
```

### 预览发布环境效果 + 静态资源分析

```
npm run report
```

### 压缩 svg 图标

```
npm run svgo
```
